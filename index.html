lastfm-mpd-cli is a program to control various functions on Last.fm through the command line. All commands operate on the current song playing in mpd (Music Player Daemon). Current available commands:
  • love - Reads the current track from mpd and 'loves' it on Last.fm

    --lastfmusername= Your last.fm username --lastfmpassword= The md5 hash of your Last.fm password --mpdpassword= Your mpd server's password --mpdhostname= The machine your server runs on --mpdport= The port your server listens on

All parameters can be specified in a configuration file as well, but will be overrided if they are specified at the command line. The global configuration can be found at /etc/lastfm-mpd-cli.rc . The global configuration is overridden if the program finds a local configuration file at $HOME/.lastfm-mpd-clirc.
